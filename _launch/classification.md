# Repo Classification

**Category:** `agent-framework`

Reasoning: agent-gorgon is not a library you import, not a standalone CLI tool, and not a research artifact. It is a set of enforcement hooks that install into an agent runtime (Claude Code) plus a discovery engine. The primary consumer is the agent runtime, not a human user at the terminal. It provides scaffolding that shapes agent behavior — that is the definition of an agent-framework artifact.
